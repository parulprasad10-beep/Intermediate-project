# Student Performance Analysis 📊

## Description

This project analyzes student performance data to understand the factors that influence final grades. It explores how variables such as study time, absences, and previous grades affect academic outcomes.

## Dataset

* Source: Kaggle (Student Performance Dataset)
* Description: The dataset contains information about students including demographic details, study habits, and academic performance. It includes three grade columns (G1, G2, G3), where G3 represents the final grade.

## Steps Performed

1. Data Cleaning

   * Checked for missing values
   * Renamed columns for better readability

2. Exploratory Data Analysis

   * Analyzed relationships between variables
   * Studied patterns in student performance

3. Visualization

   * Box plots for gender vs performance
   * Study time vs final grade
   * Absences vs performance
   * Correlation heatmap

4. Model Building

   * Applied Linear Regression to predict final grades

## Results

* Previous grades (G1 and G2) strongly influence final grade
* Students with fewer absences tend to perform better
* Study time has a moderate effect on performance
* Strong correlation exists between reading progression (G1 → G2 → G3)

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn

## Conclusion

The analysis shows that consistent academic performance and attendance play a significant role in final grades. Visualization techniques helped uncover key patterns, and the regression model demonstrated predictive relationships between variables.
